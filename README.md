# 🚀 MERN Full Stack Projects

This repository contains multiple **Full Stack Web Applications** built using the **MERN Stack (MongoDB, Express.js, React.js, Node.js)**.
It demonstrates practical implementations of full-stack development including **authentication, APIs, database integration, and modern UI development**.

---

## 🛠 Tech Stack

**Frontend**

* React.js
* JavaScript (ES6+)
* HTML5 & CSS3
* Axios
* React Router

**Backend**

* Node.js
* Express.js

**Database**

* MongoDB
* Mongoose

**Other Tools**

* Git & GitHub
* REST APIs
* JWT Authentication
* Environment Variables (.env)

## ⚙️ How to Run the Projects

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/ShubhamSingh3102/MERN-FULL-STACK-PROJECT.git
```

### 2️⃣ Go to the Project Folder

```bash
cd MERN-FULL-STACK-PROJECT
```

### 3️⃣ Navigate to a Specific Project

Example:

```bash
cd Doctor-Appointment-Booking-System
```

### 4️⃣ Install Dependencies

Backend

```bash
cd backend
npm install
```

Frontend

```bash
cd frontend
npm install
```

Admin Panel (if available)

```bash
cd admin
npm install
```

---

## ▶️ Run the Application

Backend

```bash
npm run server
```

Frontend

```bash
npm start
```

Admin Panel

```bash
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file in the backend folder.

Example:

```
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```
